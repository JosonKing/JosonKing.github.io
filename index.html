java基础_常量相加、变量相加区别

Veröffentlicht am 2018-03-29

变量相加

变量相加，会将结果转为类型较大的，若为byte、char、short，则转为int。

常量相加

常量相加，会先将结果计算出来，然后看是否在接收变量类型范围内，如果不在则报错。

javase_编译运行基础

Veröffentlicht am 2018-03-29

java编译运行注意细节

javac && java

javac 跟文件名及后缀，java 只跟文件名

文件名和类名不可以一致，建议一致

文件名和类名不一致， 生成class字节码文件名与类名一致。java命令后执行文件即类名

path && classpath

path: 使得在任意路径下可以使用javac/java命令
classpath: 使得在任意路径下可以使用.class文件

classpath 路径不能有中文

java基础_原码补码反码

Veröffentlicht am 2018-03-27

计算机在计算数值时，都是采用数据对应的二进制的补码来计算的

原码

正数的原码最高位是0，负数的原码最高位是1，其他的是数值为。

十进制数据 符号位 数值为
+7 0 0000111
-7 1 0000111

反码

正数的反码与原码相同，负数的反码与原码是符号位不变，数值为取反（1变0,0变1）

十进制数据 符号位 数值为
+7 0 0000111
-7 1 1111000

补码

正数的补码与其原码相同，负数的补码是在反码的基础上+1

十进制数据 符号位 数值为
+7 0 0000111
-7 1 1111001
