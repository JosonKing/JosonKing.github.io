java基础_常量相加、变量相加区别

Veröffentlicht am 2018-03-27

变量相加

变量相加，会将结果转为类型较大的，若为byte、char、short，则转为int。

常量相加

常量相加，会先将结果计算出来，然后看是否在接收变量类型范围内，如果不在则报错。
